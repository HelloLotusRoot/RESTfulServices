# 게시판 사이트
REST api를 사용한 게시판 사이트입니다.

## 프로젝트 정보
* 기간
  * 2022.11.12 ~ 2022.11.23

## 기술스택
* 개발언어 : JAVA 17, JSP
* 프레임워크 : Spring Boot 2.7.5
* 데이터베이스 : h2
* 디자인 툴 : bootstrap
